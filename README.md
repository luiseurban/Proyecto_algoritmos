# Comparador de Algoritmos: Cadena Media

Pequeña app Flask para comparar un algoritmo ingenuo (fuerza bruta) y uno por búsqueda local para hallar la cadena media.

Requisitos
- Python 3.8+
- Windows PowerShell (ejemplos incluidos)

Instalación y ejecución (PowerShell):

```powershell
python -m pip install -r requirements.txt
python app.py
```

Abrir `http://127.0.0.1:5000/` en el navegador. Ajusta `L` con cuidado: la fuerza bruta crece exponencialmente con `L` y el tamaño del alfabeto.
